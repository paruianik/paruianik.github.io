Grayscale Image Colorization Using Convolutional Neural Networks (CNNs)
Project Description

This project focuses on automatic convert a grayscale image into a colored image using Deep Learning. The objective was to build a model capable of converting grayscale images into realistic colored versions by learning visual patterns, hidden patterns, textures, and contextual relationships from data. Image colorization is an interesting computer vision problem because there is no single correct output for a grayscale image. The model must generate plausible colors based on learned representations.

Features

The primary features of this project are:

  • Automatically convert a grayscale image into a realistic colored image using deep learning.
  • Uses CNN architecture to extract the most useful features like edges, shapes, texture, pattern etc.
  • Leverages LAB color space where the L channel is input and concatenated AB channels are targeted output.
  • Generate visually plausible and natural looking colors based on learned patterns.
  • Supports end-to-end workflow from training dataset to final colorized output of a grayscale image.
Data Processing
Before feeding the data into a model multiple preprocessing steps were performed to standardize the input and improve training efficiency.

The steps of data preprocessing are:
  • Resized all images to a fixed resolution for consistent model input.
  • Converted BGR images to RGB images.
  • Conveted RGB images to LAB image format.
  • Split the LAB images into three channels L, A, B.
  • Concatanate the A and B channel which contains the color information.
  • Used the L channel (luminosity information) as the model input.
  • Used the concatenated AB channel as the target output.
  • Convert the images into a numpy array because keras works best with array.
  • Resized the input channels images into desired size.
  • Split the dataset into two segements for training and testing purposes.

Why this is Important ?

Because proper preprocessing ensures stable training, faster convergence, and better generalization.

Methodology

The project was approached as an image-to-image translation task. The model receives a grayscale image as input and predicts its corresponding colored version. During training, the predicted output is compared with the actual color image, and model weights are updated to minimize prediction error.


Workflow:

  • Load the image dataset.
  • Resize all images to a fixed resolution to ensure consistent input dimensions.
  • Normalize pixel values to a range for better numerical stability and faster convergence.
  • Convert original RGB images into grayscale images.
  • Use grayscale images as model inputs.
  • Use the corresponding color channel iomage as target outputs.
  • Split the dataset into training and validation sets.
  • Feed grayscale images into the CNN model during training.
  • Monitor validation loss and performance after each epoch.
  • Test the trained model on unseen grayscale images.
  • Evaluate PSNR to calculate the model's performacne.

Why this particular approach ?

CNNs are highly effective for extracting spatial features and patterns from images, making them suitable for computer vision tasks such as colorization.

Model Architecture

The model is sequestial Convolutional Neural Network (CNN) designed to learn the mapping between grayscale image internsity and the corresponding color information of the image. The architecture follows an Encoder - Decoder structure within a sequential framework.


Architecture highlight:

  • Convolution Layers: Extract low-level and high-level visual features
  • Padding Layers: Preserve spatial dimensions and retain edge information during convolution operations.
  • Batch Normalization Layers: Normalize intermediate activations, improve training stability, and accelerate convergence.
  • Pooling Layers: Reduce dimensionality while preserving key information.
  • Upsampling Layers: Restore spatial dimensions for image generation.
  • Activation Functions: Introduce non-linearity for learning complex mappings.
  • Output Layer: Produces predicted color channels.

Why CNN ?

CNNs preserve spatial locality and efficiently learn textures, edges, shapes, and contextual relationships from image data.
